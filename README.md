## Requirement:

- Rust
- NodeJS

## Install

```bash
git clone git@github.com:crackeer/tauri-markdown.git
cd tauri-markdown
npm install
cargo tauri build
```

## Development

```bash
git clone git@github.com:crackeer/tauri-markdown.git
cd tauri-markdown
npm install
cargo tauri dev
```
